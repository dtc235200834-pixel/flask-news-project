# Website Quản Lý Tin Tức Bóng Đá

## Công nghệ
Python Flask  
SQLite  
Bootstrap  

## Chức năng
- CRUD bài viết
- Upload ảnh
- Tìm kiếm
- Duyệt bài
- Thống kê

## Chạy project

python app.py

http://127.0.0.1:5000
