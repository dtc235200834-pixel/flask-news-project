# HỆ THỐNG QUẢN LÝ TIN TỨC BÓNG ĐÁ

## 1. Giới thiệu
Ứng dụng web quản lý tin tức bóng đá được xây dựng bằng Flask, hỗ trợ phân quyền người dùng và thống kê báo cáo chuyên nghiệp.

## 2. Công nghệ sử dụng
- **Backend:** Flask, Flask-SQLAlchemy (SQLite)
- **Frontend:** Bootstrap 5, FontAwesome, Chart.js

## 3. Hướng dẫn cài đặt
1. Mở Terminal tại thư mục dự án.
2. Cài đặt thư viện: `pip install flask flask-sqlalchemy`
3. Chạy ứng dụng: `python app.py`
4. Truy cập: `http://127.0.0.1:5000`

## 4. Tài khoản kiểm thử
- **Admin:** `admin` / `123` (Toàn quyền quản trị)
- **Guest:** (Không cần đăng nhập, chỉ xem tin tức đã duyệt)
